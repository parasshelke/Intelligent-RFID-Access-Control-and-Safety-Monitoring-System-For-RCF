# Intelligent-RFID-Access-Control-and-Safety-Monitoring-System-For-RCF
Intelligent RFID Access Control and Safety Monitoring System for RCF is a real-time platform developed for Rashtriya Chemicals and Fertilizers Limited Thal. It uses RFID tags to track personnel entry and exit, monitor security zones, and support emergency response for improved plant safety and security.
The Intelligent RFID Access Control and Safety Monitoring System for RCF is an advanced real-time security and personnel management platform designed for Rashtriya Chemicals and Fertilizers Limited (RCF), Thal. This system integrates RFID technology, real-time monitoring, and an intelligent command interface to provide efficient access control, personnel tracking, and emergency response capabilities within one of India’s most important fertilizer manufacturing complexes.

At the core of the system is an RFID-based access control mechanism that automatically identifies and logs every individual entering or exiting the facility. Each CISF personnel member, staff employee, and authorized visitor carries a unique RFID-enabled identification card. These cards are scanned at designated entry and exit checkpoints throughout the plant. Once scanned, the system immediately records the event and updates the individual’s status in the central database. Personnel inside the facility are marked as IN (green), while those outside are marked as OUT (red), ensuring accurate and real-time accountability of everyone present within the premises.

The system includes an interactive satellite-based map interface representing the entire RCF Thal plant. The facility is divided into eight monitored security zones, allowing operators to visualize personnel distribution across the site. Each zone dynamically displays the number of individuals currently present. Four critical operational areas—UREA, NH3, PWR, and TANK—are continuously highlighted due to their high-risk nature in a chemical manufacturing environment.

A live personnel monitoring panel provides detailed real-time information including names, identification numbers, roles, assigned zones, and entry timestamps. Security officers can perform zone-level drill-down analysis to instantly view the list of personnel present within a specific zone. This feature enhances routine monitoring and significantly improves response time during security or safety incidents.

To strengthen situational awareness, the system integrates a CCTV surveillance dashboard, displaying live camera feeds from multiple locations across the facility. This enables operators to visually verify access events and monitor activities in critical areas simultaneously.

During emergencies, the platform activates Emergency Protocol Mode, transforming the interface into a high-alert command dashboard. This mode triggers alarm indicators, evacuation monitoring tools, and a red alert theme, ensuring that security commanders can quickly assess the situation and coordinate an effective response.

Designed with a dark-mode operational interface, clear visual indicators, and reliable real-time data processing, the Intelligent RFID Access Control and Safety Monitoring System establishes a modern standard for industrial security, combining smart technology, workforce safety, and emergency preparedness into a single integrated command platform.
